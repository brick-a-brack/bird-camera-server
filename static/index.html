📚 Documentation API

GET /cameras

Liste toutes les caméras disponibles avec leur index et nom

Réponse: JSON array avec les caméras

GET /snapshot/:camera_id

Récupère un snapshot JPEG de la caméra spécifiée

Paramètres: camera_id (entier)

Réponse: Image JPEG

GET /stream/:camera_id

Diffuse un flux MJPEG en continu de la caméra spécifiée

Paramètres: camera_id (entier)

Réponse: Stream MJPEG multipart/x-mixed-replace
